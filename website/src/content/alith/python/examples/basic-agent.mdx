# Basic Agent Example

This example demonstrates how to create a simple AI agent using the Alith Python SDK with Groq's free LLM service.

## Prerequisites

- Python 3.8+
- Alith Python SDK installed
- Groq API key (free)

## Setup

<Steps>

### 1. Install Dependencies

```bash
pip install alith
```

### 2. Get Groq API Key

1. Visit [Groq](https://groq.com)
2. Sign up for a free account
3. Go to API Keys section
4. Generate a new API key
5. Copy your API key

### 3. Set Environment Variable

```bash
# Unix/MacOS
export GROQ_API_KEY="your-groq-api-key"

# Windows
set GROQ_API_KEY=your-groq-api-key
```

### 4. Create Basic Agent

```python
from alith import Agent
import os

# Get API key from environment
api_key = os.getenv("GROQ_API_KEY")

# Create agent
agent = Agent(
    name="BasicAgent",
    model="llama-3.3-70b-versatile",
    api_key=api_key,
    base_url="https://api.groq.com/openai/v1",
    preamble="You are a helpful AI assistant."
)

# Test the agent
response = agent.prompt("Hello! How are you?")
print(response)
```

</Steps>

## Complete Example

```python
from alith import Agent
import os

def main():
    # Get API key
    api_key = os.getenv("GROQ_API_KEY")
    if not api_key:
        print("Error: GROQ_API_KEY environment variable not set")
        return
    
    # Create agent
    agent = Agent(
        name="BasicAgent",
        model="llama-3.3-70b-versatile",
        api_key=api_key,
        base_url="https://api.groq.com/openai/v1",
        preamble="You are a helpful AI assistant that provides clear and accurate answers."
    )
    
    print("Basic Agent is ready!")
    print("Ask me anything (type 'quit' to exit)")
    print("-" * 50)
    
    # Interactive loop
    while True:
        try:
            user_input = input("\nYou: ").strip()
            
            if user_input.lower() in ['quit', 'exit', 'bye']:
                print("Goodbye!")
                break
            
            if not user_input:
                continue
            
            # Get response from agent
            response = agent.prompt(user_input)
            print(f"\nAgent: {response}")
            
        except KeyboardInterrupt:
            print("\nGoodbye!")
            break
        except Exception as e:
            print(f"Error: {e}")

if __name__ == "__main__":
    main()
```
